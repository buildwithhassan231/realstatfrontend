# Real Estate Frontend

A modern and responsive real estate website built with Next.js and React. This project provides a polished user experience for browsing properties, exploring agents, viewing projects, and managing listings through dedicated dashboard and admin interfaces.

## 🌟 Overview

This application is designed for a real estate platform where users can:

- Browse featured and available properties
- View detailed property information and images
- Explore real estate agents and projects
- Register or log in to access personalized experiences
- Manage listings and inquiries through agent and admin dashboards

The UI is built to be clean, professional, and easy to navigate for both visitors and administrators.

## ✨ Key Features

- Responsive homepage with hero section and featured properties
- Property listing and detail pages
- Agent profile and listing pages
- Projects showcase and interest registration
- Authentication pages for login and registration
- Agent dashboard for managing listings and inquiries
- Admin dashboard for managing users, categories, properties, and settings
- Modern styling with Tailwind CSS

## 🛠️ Tech Stack

- Next.js 16
- React 19
- Tailwind CSS
- ESLint
- JavaScript

## 📁 Project Structure

```bash
src/
  app/                # App routes and pages
  components/         # Reusable UI components
  context/            # Auth and shared state context
  data/               # Static data for listings, agents, and projects
```

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- Node.js (recommended: latest LTS version)
- npm, pnpm, yarn, or bun

### Installation

Clone the repository and install dependencies:

```bash
git clone <your-repository-url>
cd fontend
npm install
```

### Run the Development Server

```bash
npm run dev
```

Then open your browser at:

```text
http://localhost:3000
```

## 🧪 Available Scripts

Run these commands from the project root:

```bash
npm run dev      # Start the development server
npm run build    # Create a production build
npm run start    # Start the production server
npm run lint     # Run ESLint checks
```

## 🔧 Development Notes

If you want to contribute or customize the project:

1. Edit pages inside the src/app directory
2. Add or update reusable UI in the src/components directory
3. Keep the styling consistent with the existing Tailwind-based design
4. Test the app locally before pushing changes

## 📦 Deployment

This project can be deployed on platforms such as Vercel, Netlify, or any hosting service that supports Next.js applications.

For Vercel deployment, the process is straightforward:

1. Connect your GitHub repository
2. Import the project in Vercel
3. Deploy with the default Next.js settings
